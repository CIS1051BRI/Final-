Project Name: Daily Deficit
What It Does:
- Takes user input which is weight, calorie goal, calories eaten, exercise time, and difficulty level.
- Assigns a burn rate using MET values which measures the intensity of the exercise and amount of energy used in exercise.
- It then finds how many calories burned through exercise and its reflected in the progress bar
- Tells you how many calories you are over or under, or if you hit your goal
